# crumpled-tree-string

"true" と "false" の２つの文字列を記憶しても、思い出すときに "tause" という文字列は出てこないでしょう。  
名前は混ぜずに覚えているものと思います。  

↓ では次の２９文字で書かれた３つの文字列を見てください。  

```plain
transaxtion
translation
station
```

↑ これは、↓ 以下のように覚えれば １６文字で済む気もします。  

```plain
        +-- ax --+
        |        |
trans --+        +-- tion
        |        |
        +-- la --+
                 |
sta -------------+
```

もちろん分解しすぎると つながりを覚える方が大変になります。
